电商退款审核 Agent
基于 LangGraph 与 FastAPI 构建的智能退款审核服务，具备长期/短期记忆、工具调用、人工介入审查（HIL）及会话管理能力。

✨ 主要特性
FastAPI 后端服务：高性能异步 API 框架，提供 RESTful 接口。

短期记忆：使用 PostgreSQL 数据库持久化存储对话上下文，支持多轮交互。

长期记忆：支持用户维度的结构化数据读写，可记录偏好、历史决策等。

Function Calling：集成自定义工具（如订单查询、退款计算、风控检查），让 Agent 能够自主调用。

人工介入审查（HIL）：支持四种工具调用审查模式，确保关键操作需人工确认。

自动批准（低风险）

始终询问（高风险）

自动拒绝（禁止操作）

条件触发（基于规则）

会话状态管理：基于 Redis 存储用户会话状态，支持服务端/客户端故障后恢复。

动态会话过期：可调整会话有效期，适配不同业务场景。

智能会话恢复：用户登录后自动加载最近会话，无历史时自动创建新会话。

历史会话管理：提供会话列表、切换、恢复功能。

前端演示：使用 Rich 库构建美观的命令行演示，实现前后端联调。
